# Draw Line
## 出现原因

在通过**Alipay**购买基金时, 最多只能看到近三年的, 信息非常不足, 于是想通过Pyecharts 实现对收盘价进行可视化, 于是就有了这次的小工具

## 目前实现的功能:

- 对收盘价进行可视化

	> 暂时仅支持通过[英为财经](https://cn.investing.com/)网站下载的数据进行可视化
